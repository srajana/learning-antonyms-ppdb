The final files are as follows:

1) antonym-list-1.txt: Antonyms generated from PPDB paraphrase type 1 where synonyms from each word were generated using WordNet.
2) antonym-list-2.txt: Antonyms generated from PPDB paraphrase type 1 where synonyms from each word were generated using lexical paraphrases from PPDB.
3) antonym-list-3.txt: Antonyms generated from PPDB paraphrase type 2.
4) snow-paths.txt: Hearst/Snow patterns generated using antonyms from antonym-list-1.txt
5) snow-paths-morph.txt: Hearst/Snow patterns generated using antonyms from antonym-list-3.txt
6) wordnet-antonyms.txt: Direct antonyms obtained from WordNet
7) wordnet-synonyms-antonyms.txt: Direct + Indirect antonyms obtained from WordNet
8) not-antonyms.txt: PPDB antonym pairs <X,Y> got from phrases <Not X, Y>
9) exclusion-antonyms.txt: Ellie's PPDB phrases classified as 'Exclusive'

PPDB paraphrase type 1: Phrases of the form 'not happy/unhappy'
PPDB paraphrase type 2: Phrases of the form 'unhappy/..', 'demotivate/...', 'inappropriate/...' etc.
